This function reads the ingredientsdata file.

This function is used in the beginning of the foodForChange.c file to load the ingredients file in the data folder The function takes a pointer to the path of the data file as a parameter. The function also takes a pointer to an array of IngredientsData and an array of Category. These parameters are used to return the loaded ingredients and categories. The function also takes a pointer to the integers ingredientsLength and categoriesLength. These parameters are used to return the length of the two arrays.
